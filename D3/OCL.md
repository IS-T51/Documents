
# OCL
Nel presente capitolo vengono descritti i vincoli OCL delle classi.

> #### **Cronometro**
>> #### stato : Enum 
>> #### **Invarianti**:
>> - stato assume i valori “reset”, “run”, “pause”
>> #### start()
>> #### precondizioni:
>> - stato deve essere in “reset”
>> #### postcondizioni:
>> - stato deve essere in "run"
